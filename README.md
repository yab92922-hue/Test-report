# 🎬 Movie Recommendation System

A simple **Movie Recommendation System built with Python** that suggests movies based on user preferences.
This project demonstrates basic recommendation logic and a simple frontend interface.

---

## 📌 Features

* Movie recommendation based on similarity
* Simple and clean interface
* Backend built with Python
* Frontend using HTML
* Easy to run locally

---

## 📂 Project Structure

```
movie-recommendation/
│
├── front/              # Frontend files (HTML)
│   └── index.html
│
├── app.py              # Main backend application
├── requirements.txt    # Project dependencies
├── .gitignore          # Ignored files
├── README.md           # Project documentation
│
├── env/                # Environment folder (ignored)
├── venv/               # Virtual environment
└── __pycache__/        # Python cache
```

---

## 🛠 Technologies Used

* Python
* Flask
* HTML
* Pandas
* Scikit-learn

---

---

## ▶️ Run the Project

Start the application with:

```
python app.py
```


---

## 🎥 How It Works

1. User opens the web interface.
2. Selects or searches for a movie.
3. The system finds similar movies.
4. Recommended movies are displayed.
